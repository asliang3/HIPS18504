🎲 Generalized Coupon Collector Problem

Chess Pieces Edition: How many draws to complete a set?

📐 Theory

Problem: Draw chess pieces i.i.d. uniformly at random (with replacement) from $n$ types. We need $m_i$ copies of piece $i$. How many draws $T$ until every quota is satisfied?

Closed-form expected value (via Poisson embedding):

$$\mathbb{E}[T_{\mathbf{m}}] = n\int_0^\infty \left[1 - \prod_{i=1}^n \left(1 - e^{-x}\sum_{k=0}^{m_i-1}\frac{x^k}{k!}\right) \right]dx$$

Key idea: Embed draws in a rate-1 Poisson process. Thinning gives independent Poisson streams of rate $1/n$ per type, so the wait for $m_i$ copies of type $i$ is $Y_i \sim n \cdot \mathrm{Gamma}(m_i, 1)$. Then $\mathbb{E}[T] = \mathbb{E}[\max_i Y_i]$ by Wald's identity, and the max-of-independent formula plus the Gamma CDF $\Pr(X_i \le x) = 1 - e^{-x}\sum_{k=0}^{m_i-1} x^k/k!$ gives the integral above.

Special case $m_i = 1$ for all $i$ (standard coupon collector):

$$\mathbb{E}[T] = n\int_0^\infty \bigl[1-(1-e^{-x})^n\bigr]\,dx = n\,H_n$$

For $n=6$ uniform types: $\mathbb{E}[T]=6 H_6 = 6(1+\tfrac12+\cdots+\tfrac16)\approx 14.70$.

One set ($n=6$ types): $\mathbf{m} = (8,2,2,2,1,1)$ — pawns, rooks, bishops, knights, queen, king.

Two sets ($n=12$ types, white + black): same $\mathbf{m}$ vector doubled — we treat white king, black king, etc. as distinct coupon types and need one of each.

Both integrals are evaluated numerically via composite Simpson's rule.
