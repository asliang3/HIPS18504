Fibonacci Turing Machine

Double-tape TM generating Fibonacci numbers in 16-bit binary

How it works: A double-tape Turing machine computes Fibonacci numbers via binary addition. Tape 1 (Work Tape) holds two 16-bit numbers (A and B) and performs A + B → result. Tape 2 (Output Tape) stores each generated Fibonacci number. Values are capped at 16 bits (max F(24) = 46368). Use Step to advance one instruction, or Run for continuous execution.
Speed: 250 ms
State: INIT

Fib Count: 0
Status: Idle
